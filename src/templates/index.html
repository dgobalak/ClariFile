Text Extraction

We extract and clean the text content of a media file using 3rd party libraries. We support the following extensions: '.wav', '.mp3', '.mp4', '.mov', '.pdf', '.jpg', '.png', '.txt', '.jpeg', '.html'

View Code »

Keyword Identification

We identify named entities from the text using natural language processing. This is done with a chunking process using the NLTK library that identifies parts of speech.

View Code »

Wiki Summarization

We scrape Wikipedia articles using a custom web scraper. The articles are then summarized using either unsupervised learning (K-Means Clustering) or the TF-IDF algorithm.
